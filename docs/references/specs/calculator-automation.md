# Calculator Automation

For development purposes, you can use the following command to create a new scoring file:

```bash
npx vue-cli create-calculator <name>
```

This will create a new scoring file in the `src/scoring` directory and a new calculator component in the `src/components/calculators` directory.

The command will ask you for the name of the calculator, the theme, and the complexity.

The name of the calculator will be used to create the scoring file and the calculator component.

The theme will be used to create the CSS file for the calculator.

The complexity will be used to create the test file for the calculator.
